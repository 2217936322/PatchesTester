# PatchesTester

用来检测 Windows 系统尚未安装的补丁信息，并根据指定的严重等级按顺序输出到文件中，可以作为 Windows 服务器运维人员日常检查更新或者网络安全人员对 Windows 服务器安全检查的有力工具。

## 工具特点：

开发语言 ：C++

开发工具 : VS2017

适用平台 : Windows 2003 及以后

使用条件 ：要求工具运行目录有可写权限

## 使用方法：

C:\Users\Administrator\Desktop>.\PatchesTester.exe
